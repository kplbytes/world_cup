# 小红书发布指南

## 发布步骤

1. 打开小红书 App，点击「+」发布笔记
2. **先上传 4 张配图**（顺序很重要，见下方「配图」部分）
3. 复制下方「正文内容」粘贴到文案框
4. 填写标题
5. 添加话题标签（文末已列出）
6. 发布

## 正文内容（直接复制粘贴）

用 AI 预测世界杯，整个系统开源了 ⚽

FastAPI + React 19 + AI，从数据同步到赛后评分全链路跑通。48 队 72 场，不是 demo。

🔗 github.com/kplbytes/world_cup

✅ 多层预测融合
Elo+Poisson 基线 → AI 预测 → 集成融合 → 24h锁定 → 赛后 Brier 评分
AI 提示词分 v1/v2 两版，v2 不给基线参考避免抄袭，还会检测 AI 是否抄基线

✅ 真实数据校准
WC 2026 已完赛 37 场回测：
精确比分 5.4% → 13.5%（2.5倍）
±1 球 73% → 75.7%

✅ 球队画像接入预测引擎
48 队 7 模块画像（实力/攻防/战术/阵容...），FIFA 官方名单已接入。v3-profile 模型把画像转成预测调整

✅ 一键全流程
9 步工作流可手动触发：同步赛果→重算→AI预测→集成→锁定→评分
首页可看运行进度，JSON 日志带请求 ID

✅ Apple 风格前端
4 页面：今日工作台/比赛中心/模型复盘/冠军与赛程
