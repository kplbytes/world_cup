# 因子准入评审报告 - 第二轮严格验证

**评审时间**: 2026-06-15
**最终结论**: **NEEDS_MORE_DATA**

---

## 1. 执行摘要

经过第二轮严格验证，LR_full（elo_diff + 新因子）相对 EloPoisson Baseline 在验证集上 Brier 改善 1.4%，Bootstrap 95%CI 显著，Walk-Forward 8/8 窗口均优于 Baseline。

**但存在以下关键问题导致无法给予 PASS_SHADOW：**

1. **世界杯样本严重退化**
2. **新因子单独无价值**：LR_new_only（不含 elo_diff）显著差于 EloPoisson
   - 盲测集世界杯 32 场，LR_full Brier=0.897 vs EloPoisson 0.838，退化 7.0%
   - LR_new_only 验证集 Brier=0.5255，差于 EloPoisson 0.5136
4. **改善主要来自 elo_diff 的 LR 重新校准**，非新因子贡献

## 11. 最终决策

### 决策：**NEEDS_MORE_DATA**

### 决策条件

| 条件 | 结果 |
|------|------|
| LR_full 显著优于EP | ✓ 是 |
| Walk-Forward 稳定 | ✓ 是 (LR优于EP窗口: 8/8) |
| 世界杯无退化 | ✗ 否 |
| 新因子单独有价值 | ✗ 否 |
| 盲测集整体无退化 | ✓ 是 |
| 盲测集世界杯无退化 | ✗ 否 |

### 理由

1. **整体改善微弱**：验证集 Brier 相对改善 1.4%，未达 2% 准入门槛
2. **改善来源不明确**：LR_full 改善主要来自 LR 对 elo_diff 的非线性校准，而非新因子增量信息。LR_new_only 显著更差
3. **世界杯场景退化**：盲测集世界杯样本 Brier 退化，违反准入条件

### 下一步建议

1. **积累更多世界杯样本**：2026世界杯正赛（6-7月）完成后重新评估
2. **引入赔率数据**：市场因子（odds_implied_prob）覆盖率当前为0%，是潜在高价值因子
3. **引入 FIFA 排名**：fifa_rank_diff 覆盖率为0%，需补充数据
4. **简化因子集**：移除冗余因子，聚焦有贡献的因子
5. **考虑模型校准**：LR 对 elo_diff 的校准改善可能是独立价值，但需与 Platt Scaling 等纯校准方法对比
6. **增加 host_nation 标识**：当前数据无法区分东道主效应，需补充
