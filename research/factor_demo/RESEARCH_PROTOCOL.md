# 世界杯历史数据因子研究协议

## 1. 研究目标

通过历史国家队比赛数据，构建预测比赛胜、平、负所需的赛前因子，并严格验证每个因子的真实性、稳定性和增量价值。

## 2. 数据范围

- 时间范围：2010年1月1日 至 当前
- 赛事类型：世界杯及预选赛、各洲杯及预选赛、欧国联、国际友谊赛
- 数据来源：Kaggle International Football Results (1872-2025)、World Football Elo Ratings、FIFA Ranking
- 友谊赛权重降低，正式比赛权重正常

## 3. 目标变量

- 主队胜 (H)、平局 (D)、客队胜 (A)
- 仅统计90分钟常规时间结果
- 点球大战不计入90分钟结果，视为平局
- 加时赛结果不计入90分钟结果，以加时赛前比分判定

## 4. 评估指标

### 主要指标
- **Brier Score**: `1/N * Σ(p_i - o_i)²`，三项概率分别计算后取平均
- Brier Score 越低越好

### 辅助指标
- **Log Loss**: `-1/N * Σ(o_i * log(p_i))`
- **概率校准**: 预测概率与实际频率的偏差（校准曲线、ECE）
- **命中率**: 仅作辅助参考，不作为准入标准

### 准入门槛
- Brier 相对改善 ≥ 2%
- Log Loss 无明显恶化（相对恶化 < 1%）
- 概率校准无明显恶化
- 至少两个独立时间窗口表现不劣于 Baseline
- 世界杯样本上不出现明显退化

## 5. 时间边界

### 训练集
- 2010-01-01 至 2018-12-31

### 验证集
- 2019-01-01 至 2021-12-31

### 最终测试集（模型确定前不得查看）
- 2022-01-01 至 2025-12-31
- 包含 2022 世界杯（2022-11-20 至 2022-12-18）

### 滚动验证
- 起始训练窗口：2010-01-01 至 2016-12-31
- 滚动步长：1年
- 每步预测下一年度所有比赛

## 6. 数据泄漏禁止清单

以下字段/数据在计算赛前因子时**严禁使用**：

| 禁止字段 | 原因 |
|---------|------|
| 本场比赛比分 | 未来数据 |
| 本场比赛进球数 | 未来数据 |
| 本场比赛射门/控球等统计 | 未来数据 |
| 赛后更新的 FIFA 排名 | 使用赛前最新排名 |
| 赛后更新的 Elo 评分 | 使用赛前计算的 Elo |
| 本场比赛赔率变化（赛中） | 未来数据 |
| 赛后新闻/评论 | 未来数据 |
| 最终赛事排名/出线结果 | 未来数据 |
| 点球大战结果 | 非90分钟数据 |
| 加时赛进球 | 非90分钟数据 |

## 7. 研究原则

1. **时间一致性**: 每场比赛只能使用开赛前已公开的数据
2. **禁止随机划分**: 必须按时间回放，禁止随机划分训练/测试集
3. **样本量要求**: 不能只使用世界杯比赛，样本量过小
4. **友谊赛降权**: 正式比赛为主，友谊赛降低权重
5. **Brier 优先**: Brier Score 为主要指标，命中率只作辅助
6. **完整报告**: 研究失败也必须输出结果，禁止只报告有效因子
7. **可复现性**: 同一输入必须生成完全一致的特征

## 8. 因子准入流程

1. 单因子研究 → 2. 增量价值验证 → 3. 消融实验 → 4. 滚动回测 → 5. 准入评审

未通过准入的因子标记为 `rejected` 或 `needs_more_data`，不得接入主程序。
